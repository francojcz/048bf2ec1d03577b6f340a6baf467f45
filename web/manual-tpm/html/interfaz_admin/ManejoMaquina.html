Manejo de equipos ó máquinas


MANUAL DE USUARIO
 TPM-QLabs  - MANEJO EQUIPOS


Cuando el usuario selecciona del menú la opción Equipos, el sistema le desplegará una pantalla como la que se muestra en la Figura 1.  En esta pantalla el usuario podrá crear, actualizar, eliminar, visualizar y restablecer un equipo.  Además, podrá registrar los periodos de mantenimiento de cada equipo.
Figura 1. Pantalla de manejo de equipos.
Agregar equipo
Para agregar un equipo pulse el botón Agregar, diligencie el formulario que se muestra al lado derecho de la pantalla y pulse el botón Guardar.  La información que se debe ingresar es la siguiente:
  • Código de inventario:  Digite el código con el que se identificará el equipo en su inventario  (es un campo obligatorio).
  • Nombre equipo:  Digite el nombre del equipo  (es un campo obligatorio).
  • Estado:  Seleccione el estado del equipo  (es un campo obligatorio).
  • Tiempo inyección (Min.):  Digite el tiempo de inyección estándar del equipo  (no es un campo obligario, por defecto es 0). Este campo se tiene en cuenta para el calculo de algunos indicadores en los reportes, por tanto es necesario que sea diligenciado con información veridica y precisa.
  • Marca:  Digite la marca del equipo (es un campo obligatorio).
  • Modelo:  Digite el modelo del equipo (es un campo obligatorio).
  • Fecha de adquisición:  Seleccione o digite en formato (AAAA-MM-DD) la fecha de adquisión del equipo  (es un campo obligatorio).
  • Computador: Seleccione el nombre del computador (nombre de certificado del computador) en el cual se registrarán los datos (es un campo obligatorio).
  • Grupo equipo: Seleccione el grupo al cual pertenece el equipo (es un campo obligatorio).
  • Cálculo indicadores: Seleccione Activo si el equipo debe tenerse en cuenta a la hora de realizar el cálculo de los indicadores correspondientes a TPM.  En caso de que el equipo deba tenerse en cuenta solo para el manejo de inventario y mantenimientos, seleccione Inactivo (es un campo obligatorio).

Actualizar equipo
Para actualizar un equipo, seleccione de la lista el equipo que desea actualizar, ingrese la información en el formulario ubicado al lado derecho y pulse el botón Actualizar. El sistema desplegará una pequeña ventana donde le preguntará la razón por la cual desea actualizar el equipo, digite la razón y pulse el botón Aceptar.  Luego el sistema le mostrará un breve mensaje en el que informa del éxito del cambio.


Eliminar equipo

Para eliminar un equipo, seleccione el equipo que desea eliminar de la lista y pulse el botón Eliminar.  El sistema desplegará una pequeña ventana donde le preguntará si realmente desea eliminar el equipo y porque razón desea eliminarlo, digite la razón y pulse el botón Aceptar. Luego el sistema le mostrará un breve mensaje en el que informa del éxito de la eliminación.


Restablecer equipo

Si usted ha eliminado un equipo, tiene la posibilidad de recuperar la información.  Para restablecer un equipo, seleccione la opción para ver los equipos eliminados (estrella blanca), seleccione de la lista el equipo que desea recuperar y pulse el botón Restablecer, luego el sistema le preguntará la razón de porqué usted desea restablecerlo, digite la razón y pulse el botón Aceptar.  Luego el sistema le mostrará un breve mensaje en el que le informa si se ha restablecido con éxito.


Periodo Mantenimiento

Para registrar los periodos en los cuales debe realizarse mantenimiento a un equipo, seleccione el equipo al cual desea asignarle un periodo (1) y luego pulse el botón Periodo Mantenimiento (2), tal y como se muestra en la Figura 2.
Figura 2. Pantalla para la asignación de periodos de mantenimiento de equipos.


Cuando el usuario selcciona el equipo y pulsa el botón Periodo Mantenimiento, el sistema despliega un a ventana como la que se muestra en la Figura 3, en donde debe seleccionar en la primera lista el periodo de mantenimiento que desea registrar.

Figura 3. Pantalla para la selección de periodos de mantenimiento.


Una vez seleccionado el periodo de mantenimiento, ingrese la fecha a patir de la cual comenzará a realizar el mantenimiento al equipo (1) seleccionando alguna fecha disponible en el calendario (2) que se despliega después de pulsar el botón para el ingreso de la fecha.  Para finalizar el proceso de registro, debe pulsar el botón Agregar periodo (3), tal y como se muestra en la Figura 4.

Figura 4. Pantalla para la selección de la fecha de inicio del periodo de mantenimiento.


Para eliminar un periodo de mantenimiento asignado a algún equipo, seleccione de lal la lista de periodos registrados el periodo que desea eliminar (1) y pulse el botón Eliminar periodo (2), tal y como se muestra en la Figura 5.  El usuario debe ser consciente de que esta información una vez eliminada no se podrá recuperar.


Figura 5. Pantalla para la eliminación de un periodo de mantenimiento.
