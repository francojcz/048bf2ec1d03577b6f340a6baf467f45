Manejo de información de Mantenimiento Autónomo


MANUAL DE USUARIO
 TPM-QLabs  - MANEJO DE MANTENIMIENTO AUTÓNOMO


Cuando el usuario selecciona del menú la opción Mantenimiento, el sistema le desplegará una pantalla como la que se muestra en la Figura 1.  En esta pantalla el usuario podrá crear, actualizar, eliminar y visualizar la información corresponde al mantenimiento de cada equipo.
Figura 1. Pantalla de manejo de mantenimiento autónomo.

Agregar Mantenimiento
Para agregar un equipo seleccione el grupo al cual pertence el equipo desde la lista Grupo equipo (1) , luego seleccione el Equipo (2) sobre el cual desea realizar el registro y por último para visualizar los registros realizados en una fecha específica seleccione Fecha de Registro (3).  Una vez seleccionado los anteriores campos, se muestran las partes o repuestos que corresponden al equipo seleccionado (4), tal y como se muestra en la Figura 2.
Figura 2. Pantalla para agregar mantenimientos.
Una vez se encuentran disponibles las partes del equipo, para agregar un registro correspondiente al mantenimiento de un equipo debe seleccionar todas las partes (1) o al menos una de las partes (2) que conforman el equipo seleccionado, luego debe pulsar el botón Agregar Parte (3) y el sistema realizará el registro del cambio del repuesto (4).  De cada registro se debe ingresar la cantidad consumida del repuesto (5), la fecha en la cual se realiza el cambio de la parte (6) y se mostrará la fecha en la cual deberá cambiar nuevamente la parte (7) la cual se calcula a partir de la periodicidad de cambio ingresada en el manejo de repuestos y se debe ingresar alguna observación (8) resultante del proceso de cambio del repuesto.  La información que debe ingresar de cada registro se encuentra resaltada con color amarillo, tal y como se muestra en la Figura 3.
Figura 3. Pantalla para agregar partes.
 

Eliminar Mantenimiento

Para eliminar un registro correspondiente al mantenimiento de un equipo, seleccione el regristo que desea eliminar de la lista (1) y pulse el botón Eliminar (2).  El sistema desplegará una pequeña ventana donde le preguntará si realmente desea eliminar el registro equipo en donde debe pulsar el botón Si (3).  Luego el sistema le mostrará un breve mensaje en el que informa del éxito de la eliminación.


Figura 4. Pantalla para eliminar partes.
